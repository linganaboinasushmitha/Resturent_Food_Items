# 🍽️ Restaurant Food Items App

A full-stack Restaurant Food Items web application built using React and Node.js.  
Users can view food items dynamically fetched from the backend API.

---

## 🚀 Live Demo

🔗 **Frontend (Live Website):**  
https://resturent-food-items-3.onrender.com  

🔗 **Backend API:**  
https://resturent-food-items.onrender.com  

---

## 🛠️ Tech Stack

### Frontend
- React
- Axios
- React Router
- Framer Motion
- CSS

### Backend
- Node.js
- Express.js
- MongoDB

---

## ✨ Features

- Dynamic food item listing
- Backend API integration
- Responsive UI
- Full-stack deployment on Render
- Clean component-based architecture
